# FitPlan Lite 演示指南

## 🎯 快速体验

### 1. 启动应用
```bash
# 确保 PostgreSQL 已启动
./start.sh
```

### 2. 访问应用
- 前端应用：http://localhost:3000
- 后端API：http://localhost:8000/docs

## 📋 功能演示

### 1. Dashboard 概览页面
- 查看本周完成率进度条
- 浏览训练统计数据
- 查看每周进度条（7天状态）

### 2. 创建训练计划
1. 点击 "创建/编辑计划"
2. 选择训练日和休息日
3. 为训练日选择预设模板或自定义
4. 添加训练动作
5. 保存计划

### 3. 日程管理
- 日历视图：直观查看月度训练安排
- 列表视图：详细查看每天训练计划
- 点击日期查看训练日详情

### 4. 完成打卡
1. 进入训练日详情页面
2. 点击 "标记为完成" 按钮
3. 查看Dashboard上的完成率变化

### 5. 生成下周模板
1. 在Dashboard点击 "生成下周模板"
2. 查看基于本周表现生成的模板
3. 点击 "应用模板" 创建下周计划

### 完成率计算

- 公式：完成训练日数 / 总训练日数

## 🎓 使用技巧

### 1. 如何创建高效训练计划
- 每周至少安排1天休息
- 训练类型搭配合理（Push/Pull/Legs）
- 动作数量适中（3-4个/天）

### 2. 如何提高完成率
- 固定训练时间
- 制定可实现的目标
- 记录训练感受

### 3. 模板生成策略
- 完成率≥80%：增加训练强度
- 完成率50-80%：保持当前强度
- 完成率<50%：降低训练强度

## 🐛 常见问题

### Q: 如何修改已创建的计划？
A: 点击 Dashboard 上的 "创建/编辑计划" 按钮，修改后保存。

### Q: 休息日可以打卡吗？
A: 可以打卡，但不计入完成率计算。

### Q: 模板生成的依据是什么？
A: 基于本周的训练完成率和训练日数量。

### Q: 如何删除计划？
A: 目前版本不支持删除计划，可直接修改现有计划。
